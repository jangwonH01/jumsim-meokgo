## 마일스톤별 기능

### 🟢 v1.0 (출시 완료, 2026-05-31)

핵심 흐름: 룰렛 / 근처 식당 / 팀 투표

- 룰렛 메뉴 추첨
- 근처 식당 검색 (카카오 Local API)
- 후보 담기 (localStorage)
- 팀 투표 생성 + 실시간 (Firestore)
- 토스 공유 (getTossShareLink)

### 🟢 v1.1 (구현 완료, 검수 대기)

핵심: 투표 결정 자동화

- 투표 종료 시간 선택 (5/15/30분)
- 카운트다운 + 자동 종료
- 동률 시 자동 결선 투표 (2분)
- 결선도 동률 시 자동 룰렛
- 결과 확정 화면 (vote/runoff/roulette 별 연출)

### 🟢 v1.2 (구현 완료, 검수 대기)

핵심: 메뉴 → 식당 → 길찾기 연결

- 메뉴 확정 후 근처 식당 자동 추천 (카카오 keyword 검색)
- 검색 반경 칩 (500m/1km/2km)
- 카카오맵 장소 외부 링크
- 카카오맵 길찾기 외부 링크

## 🟡 v1.3 (다음 — 1~2개월 내)

핵심: 사용자 개인화

### 후보 기능

1. **자주 가는 식당 즐겨찾기**
   - 근처 검색 결과에서 ⭐ 즐겨찾기
   - 별도 즐겨찾기 화면
   - 다음 검색 시 우선 노출

2. **자주 먹는 메뉴 즐겨찾기**
   - 룰렛 화면에 본인 메뉴 12개 + 즐겨찾기 추가
   - 빠른 룰렛 (즐겨찾기만으로)

3. **투표 이력 보기**
   - 지난 결정들 모음 (홈에 진입점)
   - "지난주 가장 많이 먹은 메뉴" 통계

4. **위치 권한 거부 시 수동 위치**
   - 주소 검색 (카카오 keyword) 으로 수동 위치 설정
   - 권한 없어도 근처 식당 검색 가능

### 기술 작업

- localStorage 키 추가 (`favorites.restaurants`, `favorites.menus`, `history`)
- 새 화면 1~2개 추가
- 카카오 주소 검색 API 통합

### 예상 작업량

- 개발: 1~2주
- 검수: 1주
- 출시: 약 1개월 후

## 🟡 v1.4 (3~6개월)

핵심: 푸시 알림 + 토스 통합

### 후보 기능

1. **푸시 알림**
   - 점심시간 30분 전 알림 ("오늘 점심 정했어요?")
   - 투표 응답 알림 (생성자에게)
   - 토스 알림 / 푸시 SDK 통합

2. **토스 로그인 통합 (선택적)**
   - 익명 → 로그인 시 디바이스 간 동기화
   - 즐겨찾기/이력 클라우드 백업
   - 토스 로그인 약관 등록 필요

3. **검색 (메뉴 키워드)**
   - 룰렛에 "찜닭" 입력 → 카카오 검색 → 후보 자동 추가

### 기술 작업

- mTLS 인증서 발급 (사업자만 가능)
- 토스 로그인 약관 등록
- Firestore users 컬렉션 추가
- 푸시 알림 SDK 통합

### 예상 작업량

- 개발: 2~3주
- 검수: 1주

## 🔵 v2.0 (6개월 ~ 1년)

핵심: 수익화 + 커뮤니티

### 후보 기능

1. **B2B 식당 광고 (사용자 1,000+ 이후)**
   - 식당 사장님이 자기 식당 광고 등록
   - 점심먹Go 검색 결과 상단 노출
   - 월 1~5만원 구독료

2. **앱인토스 배너 광고**
   - 결과 화면 하단 배너 1개
   - 사용자 경험 영향 최소

3. **배달앱 딥링크**
   - 배민 / 쿠팡이츠 / 요기요 외부 링크
   - 메뉴 확정 → "배달로 주문" 버튼

4. **점심먹Go 팀 (지속 그룹)**
   - 매번 카톡 안 보내고 정해진 팀과 투표
   - 토스 로그인 + Firestore teams 컬렉션

### 예상 수익 (DAU 1,000+ 가정)

- 광고: 월 5만원~10만원
- B2B 광고: 월 10만원~30만원 (식당 3~10개)
- 합계: 월 15만원~40만원

---

## 🔵 v3.0 (1년 ~ 2년)

핵심: 데이터 사업

### 가능성

1. **트렌드 데이터 판매**
   - 지역별 점심 메뉴 인기 순위
   - 시기별 트렌드 (계절, 요일)
   - 식음료 회사 / 외식업 / 부동산 대상

2. **점심먹Go 프리미엄**
   - 광고 제거
   - 통계 / 리포트
   - 월 1,000~3,000원
   - 결제율 1% 가정: DAU 5,000 × 1% × 2,000원 = 월 10만원

3. **Go 시리즈 통합**
   - 달리GO, 출근하GO, 미팅GO 등과 통합 사용자 베이스
   - 크로스 마케팅
