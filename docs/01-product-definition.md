# 점심먹Go — 제품 정의서

> 작성일: 2026-06-05
> 버전: v1.2 (0.1.3)
> 상태: 토스 미니앱 정식 출시 (사업자 워크스페이스)

## 한 줄 정의

**"점심 메뉴 결정에 매번 30분 쓰지 않아도 되는 미니앱"** — 룰렛·근처식당·팀투표·자동 식당 추천으로 5분 안에 결정.

## 핵심 가치 (Value Proposition)

| 사용자 페인 포인트 | 점심먹Go 의 해결책 |
|---|---|
| "오늘 뭐 먹지" 매일 30분 고민 | 🎰 룰렛 3초 결정 |
| 동료들과 메뉴 못 정함 ("아무거나" 무한루프) | 🗳️ 실시간 팀 투표 30초 |
| 처음 가는 동네에서 식당 모름 | 📍 카카오맵 기반 근처 식당 검색 |
| 투표 동률이면 또 못 정함 | ⚖️ 자동 결선 투표 → 🎰 룰렛 |
| 메뉴 정해도 어디 가야 할지 모름 | 🗺️ 메뉴 확정 시 근처 식당 + 길찾기 자동 추천 |

## 타겟 사용자

| 페르소나 | 특성 | 주요 사용 시나리오 |
|---|---|---|
| 🏢 **회사원 (메인)** | 점심시간 12:00~13:00 결정 압박 | 팀 4~6명과 빠른 메뉴 결정 |
| 🎓 **대학생** | 학교 근처 식당 선택 | 친구들과 룰렛 + 근처 식당 |
| 🚙 **출장자** | 모르는 동네 식사 | 위치 기반 식당 검색 |
| 🏠 **1인 가구** | 메뉴 결정 장애 | 룰렛 단독 사용 |

## 차별점 (Competitive Edge)

| 경쟁 서비스 | 약점 | 점심먹Go 강점 |
|---|---|---|
| 카카오맵 / 네이버지도 | 식당만 보여줌, 결정 도움 X | 룰렛·투표로 결정까지 |
| 배달앱 (배민·요기요) | 배달 전제, 매장 식사 X | 매장 + 배달 모두 |
| 무료 룰렛 앱 | 메뉴/식당 연결 X | 메뉴 → 식당 → 길찾기 통합 |
| 단톡방 투표 | 별도 도구 필요 | 토스 안에서 30초 |

## 제외 (의도적으로 안 한 것)

- ❌ 회원가입/로그인 — 토스 익명 + getAnonymousKey
- ❌ 음식 리뷰 시스템 — 운영 부담 + 정책 리스크
- ❌ 사진 업로드 — 보안 + 운영 부담
- ❌ 결제/주문 — 외부 사이트(배달앱) 위임
- ❌ 광고 — DAU 1,000+ 까지 무료 운영

## 비즈니스 모델 (장기)

| 단계 | 모델 | 시점 |
|---|---|---|
| 현재 ~ DAU 1,000 | **무료** (사용자 신뢰 우선) | 출시 ~ 6개월 |
| DAU 1,000+ | 앱인토스 배너 광고 | 6~12개월 |
| DAU 5,000+ | B2B (식당 광고, 배달앱 제휴) | 1년+ |
| DAU 10,000+ | 데이터 사업, 프리미엄 | 1~2년+ |
